# Лабораторная 4: Выборки с возвращением и без, размещение m = n

## Условие

- **n** — размер исходного множества (элементы 1…n).
- **m** — количество выбираемых элементов (размер выборки).
- Параметры **n** и **m** задаются в диапазоне от 3 до 10.
- **Матрица m×n:** строка i — i-й выбор, столбец j — элемент j; в ячейке 1, если на этом шаге выбран элемент j.

## Режимы

1. **С возвращением** — m раз выбираем один из n элементов (повторы возможны).
2. **Без возвращения** — выбираем m различных элементов из n (m ≤ n).
3. **Размещение m = n** — перестановка всех n элементов (m = n).

## Запуск

```bash
cd lab-4
make          # вывод 12 таблиц в консоль
make out      # вывод в output.txt
```

Или напрямую: `python3 lab-4.py`.
