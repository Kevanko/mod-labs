# Графы по формулам

По одной папке на каждую формулу (параметр); в каждой папке — **50 картинок** графов.

- **exp/** — формула P = exp(-a·d²):
  - `0.01/`, `0.06/`, `0.15/`, `0.4/` — по 50 графов (файлы `seed_*.png`)
- **pow/** — формула P = 1/d^b:
  - `0.3/`, `0.9/`, `1.6/`, `2.5/` — по 50 графов (файлы `seed_*.png`)

Метрики по всем графам: `exp/metrics.csv`, `pow/metrics.csv`.

Из каждой папки выбран один граф (медианный по средней длине ребра) → сохранён в `images/interesting/`.
